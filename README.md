# Capta+ - Gestão de Editais

Sistema de gestão inteligente de editais e captação de recursos para instituições públicas e privadas.

## 🚀 Tecnologias

- **Next.js 14** - Framework React
- **TypeScript** - Tipagem estática
- **CSS Customizado** - Flexbox e CSS Grid
- **Lucide React** - Ícones

## 📦 Instalação

```bash
# Instalar dependências
npm install

# Nota: Após instalar as dependências, os erros de lint do TypeScript desaparecerão
# Os erros aparecem porque o TypeScript precisa dos tipos do React que vêm de @types/react

# Executar em desenvolvimento
npm run dev

# Build para produção
npm run build

# Executar produção
npm start
```

## 🎨 Funcionalidades

- ✅ Dashboard com visão geral
